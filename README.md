# 📸 Instagram Clone - Interface de Login

Este projeto é uma reprodução fiel da página de login do Instagram. O objetivo principal foi exercitar a capacidade de replicar um layout profissional existente, focando em detalhes de posicionamento, tipografia e responsividade.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação de formulários, containers de banners e links de navegação.
* **CSS3:** * **Flexbox:** Utilizado para alinhar o banner lateral e o formulário de login.
    * **Positioning:** Uso de `position: relative` e `absolute` para criar o efeito de linha divisória com texto sobreposto.
    * **Media Queries:** Interface totalmente adaptável para dispositivos móveis, escondendo o banner lateral e ajustando larguras.
    * **Hover Effects:** Transições suaves nos botões e links.

## 🧠 Desafios e Aprendizados:
* **Replicação de UI:** Foco em manter as proporções, cores (hexadecimais específicos) e fontes da interface original.
* **Layout Adaptativo:** Implementação de lógica mobile-first para garantir que o formulário ocupe 100% da tela em smartphones.
* **Componentização Visual:** Criação de caixas de formulário separadas do container de registro, mantendo a organização do código.

## ⚙️ Como visualizar
1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
